# A股量化系统整合优化计划

## 目标
把当前分散的选股、AI荐股、股票监控、模拟交易功能收敛到一套可复用的策略与风控信号体系：更科学的多因子选股、持仓过程中的加仓/减仓/平仓信号、以及页面之间一致的结果表达。

## 成功标准
- 页面导航稳定，不再出现点击后 Page not found。
- 选股、AI荐股、股票监控、模拟盘至少复用同一套策略信号结果。
- 策略信号包含买入、持有、观察、加仓、减仓、止盈、止损/平仓等动作。
- 关键策略逻辑有快速测试或导入检查。
- 系统保留“研究学习用途，不保证盈利”的边界说明。

3. [complete] 实现统一策略信号模块，并接入选股、监控、模拟盘。
4. [complete] 修复导航与页面可用性问题。
5. [complete] 运行测试和页面导入检查，记录剩余风险。

7. [complete] 研究并回测筛选策略，设计一键推荐当前候选股。
8. [complete] 优化模拟盘页面流程和下单交互。
9. [complete] 研究缠论并实现缠论AI Agent、交易策略和回测。
10. [complete] 建立规范化知识库，覆盖缠论、A股规则、量化因子、技术指标、回测风控和Agent使用规范。

## 约束
- 不做大规模重构，优先复用现有 factors、ranking、paper_trade。
- 不承诺收益，只提供可回测、可解释、可风控的决策辅助。
- 前端只做必要集成和体验修补，避免无关视觉重做。
