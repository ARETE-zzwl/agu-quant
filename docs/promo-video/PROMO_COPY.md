# TradingAgents-Astock 宣传文案

## 核心定位

**主标语：** 把 A 股研究，从一个答案，变成一场可追溯的协作。

**副标语：** 7 位 AI 分析师，多空辩论，风险复核；从实时行情到量化因子、模拟执行与每日复盘，一套本地运行的 A 股研究工作台。

## 30 字短文案

7 位 AI 分析师协作研判 A 股，让行情、政策、资金、因子与风险进入同一条研究链路。

## 100 字产品简介

TradingAgents-Astock 是面向 A 股深度特化的多 Agent 投研框架。市场、情绪、新闻、基本面、政策、游资和解禁 7 位分析师独立研究，再通过多空辩论、研究经理裁决与三方风险复核生成结构化报告。系统同时提供实时行情、板块分析、一键选股、293 因子、策略回测、股票监控、模拟盘、基金中心和每日投研报告，并支持本地部署与自备模型密钥。

## 官网 / README 首屏文案

### 让投资研究拥有过程，而不只是一句结论

传统 AI 往往直接给答案。TradingAgents-Astock 把研究拆成可以检查的协作流程：7 位分析师从不同角度收集证据，多头与空头公开辩论，研究经理形成裁决，风险角色再从激进、中性与保守视角复核。

围绕这条研究主线，系统把 A 股实时行情、板块轮动、多条件选股、293 个量化因子、策略回测、股票监控、T+1 模拟盘、基金研究和每日自动报告放进同一个本地 Web 工作台。

它不承诺收益，也不替你做决定。它做的是让每个结论有来源、每个分歧能被看见、每次执行有风险边界。
