# 宣传视频分镜与配音稿

## 视觉方向

- 画幅：16:9，1280×720，30fps。
- 调性：深色研究终端，橙色用于操作与关键动作，青绿色用于数据来源和可信感。
- 节奏：真实页面停留 5–10 秒；不使用虚构收益曲线，不快速闪烁关键风险文字。
- 配音：中文普通话，克制、清晰、偏技术产品纪录片语气。

## 总览片：TradingAgents-Astock（约 70 秒）

| 段落 | 实际画面 | 配音 |
|---|---|---|
| 1 | 品牌开场卡 | 研究 A 股，难的不是没有信息，而是把行情、政策、资金、风险和执行，放进同一套判断链路。 |
| 2 | 大盘看盘 → 一键选股结果 | TradingAgents-Astock 面向 A 股，把实时行情、板块轮动和多条件选股，直接带进本地 Web 工作台。 |
| 3 | 深度分析首页 | 输入六位代码或中文股票名，市场、情绪、新闻、基本面、政策、游资和解禁，七位分析师分别展开研究。 |
| 4 | 结构化报告 | 多头与空头交叉辩论，研究经理形成裁决，再由风险角色复核，最终生成结构化报告与 PDF。 |
| 5 | 因子引擎 | 二百九十三个因子、十大分类、回测与策略调权，让主观观点进入可检验的数据框架。 |
| 6 | AI 荐股 → 股票监控 → 模拟盘 | 从 AI 候选到多周期监控，再到遵守 T 加一规则的模拟盘，研究结果可以继续进入执行和复盘。 |
| 7 | 基金中心 → 每日投研报告 | 基金筛选、组合管理和每日投研报告，进一步覆盖跨品种研究与定时自动化。 |
| 8 | 品牌结束卡 | 核心源码开源，支持本地部署和自备模型密钥。TradingAgents-Astock，让每个结论都有过程，让每次决策都有边界。仅供学习研究，不构成投资建议。 |

## 模块片一：多 Agent 深度投研（约 30 秒）

| 段落 | 实际画面 | 配音 |
|---|---|---|
| 1 | 模块开场卡 | 不是让一个模型直接猜答案，而是让不同角色先各自把证据说清楚。 |
| 2 | 深度分析首页 | 七位分析师覆盖技术、情绪、新闻、基本面、政策、游资和解禁，并支持中文股票名称自动解析。 |
| 3 | 历史结构化报告 | 多空辩论之后，由研究经理裁决，再经过风险复核，形成包含论据、评级、行动边界和风险提示的完整报告。 |
| 4 | 模块结束卡 | 让结论可追溯，让分歧可检查。TradingAgents-Astock 多 Agent 深度投研。 |

## 模块片二：行情与量化研究（约 40 秒）

| 段落 | 实际画面 | 配音 |
|---|---|---|
| 1 | 大盘看盘 | 从大盘状态、涨跌家数与资金信号开始，先确定市场所处的环境。 |
| 2 | 板块分析 | 行业、概念和涨跌热力图，把板块轮动变成可阅读的排名。 |
| 3 | 一键选股真实结果 | 通过市场、估值、质量、市值、换手率和涨跌幅条件，快速缩小研究范围。 |
| 4 | 因子引擎 | 二百九十三个因子覆盖价值、动量、质量、资金、波动、情绪、技术与复合联动，并提供回测和策略调权。 |
| 5 | 股票监控 | 最后用多周期 K 线、因子趋势和策略信号持续监控，而不是只看一次性结果。 |

## 模块片三：模拟执行与自动化（约 38 秒）

| 段落 | 实际画面 | 配音 |
|---|---|---|
| 1 | AI 荐股 | 预设策略或自定义权重先产生研究候选，并明确结果只用于研究和模拟盘。 |
| 2 | 模拟盘 | 模拟盘遵守 A 股交易时段、T 加一和持仓规则，资产、委托与信号可以延续保存。 |
| 3 | 基金中心 | 基金中心将场外基金、ETF、组合权重和独立模拟舱放进统一界面。 |
| 4 | 每日投研报告 | 自选股可以按简报、完整或风险模板生成 Markdown 报告，并交给 Windows 任务计划定时执行。 |
| 5 | 模块结束卡 | 从候选、监控到模拟和日报，让研究工作流真正闭环。仅供学习研究，不构成投资建议。 |

## 剪辑备注

- 一键选股画面使用真实接口返回的结果，但旁白只描述筛选过程，不强调具体股票。
- 历史报告画面仅作为结构展示，不放大具体买卖评级。
- 基金列表存在接口口径差异，镜头重点放在筛选条件与多标签管理，不宣传具体区间收益。
- 发布前可将离线配音替换为商业级声音；文件名保持一致即可重新合成。
