# 量化因子体系

## 用途

为一键选股、AI荐股、持仓监控和策略调权提供统一因子分类。因子不是孤立信号，必须考虑数据质量、市场状态、相关性、换手和交易成本。

## 因子分类

| 类别 | 常见指标 | 逻辑 | A股注意点 |
| --- | --- | --- | --- |
| 价值 | PE、PB、PS、股息率、EV/EBITDA | 低估值回归 | 低估可能是基本面恶化 |
| 品质 | ROE、毛利率、现金流、负债率 | 好公司长期更稳 | 财报频率低，需防滞后 |
| 动量 | 20/60/120日收益、均线强度 | 趋势延续 | A股短期追涨风险高 |
| 反转 | 过度下跌、偏离均线、RSI低位 | 情绪修复 | 弱势股可能继续走弱 |
| 低波 | 波动率、最大回撤、下行波动 | 风险补偿和稳健收益 | 牛市中可能跑输 |
| 流动性 | 成交额、换手率、价量弹性 | 降低冲击成本 | 过高换手也可能代表投机 |
| 资金流 | 主力资金、北向、龙虎榜 | 资金偏好 | 数据口径噪声大 |
| 情绪 | 涨停强度、板块热度、新闻情绪 | 短线风险偏好 | 容易过拟合 |

## 组合原则

1. 先定股票池：剔除停牌、ST、极低流动性、高风险标的。
2. 再算横截面分位：不同量纲因子不要直接相加。
3. 控制相关性：高相关因子不要重复加权。
4. 使用滚动回测调权：市场风格会变化。
5. 加入风控过滤：高分股票如果处于跌停、巨幅放量破位，也不应直接推荐。

## 推荐默认组合

| 策略 | 权重结构 | 适用场景 |
| --- | --- | --- |
| 价值低波 | 价值30%、品质25%、低波25%、流动性20% | 防守和震荡市 |
| 成长动量 | 动量35%、品质25%、资金20%、板块强度20% | 强趋势环境 |
| 均衡多因子 | 价值20%、品质20%、动量20%、低波20%、资金20% | 默认一键推荐 |
| 反转修复 | 反转30%、低波25%、资金20%、价值15%、流动性10% | 超跌修复 |

## 验证指标

- IC均值与IC_IR。
- 分组收益单调性。
- 年化收益、最大回撤、夏普、胜率。
- 换手率和成本后收益。
- 不同年份、牛熊震荡阶段的稳定性。
