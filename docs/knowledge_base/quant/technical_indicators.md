# 技术指标体系

## 用途

技术指标用于描述价格、成交量和波动状态。它们适合做确认和过滤，不适合单独作为高确定性买卖依据。

## 趋势类

| 指标 | 解释 | 用法 |
| --- | --- | --- |
| MA/EMA | 均线或指数均线 | 判断趋势方向和支撑压力 |
| MACD | 快慢均线差与柱体 | 判断动量、背离、缠论力度近似 |
| ADX | 趋势强度 | 区分趋势和震荡 |

## 震荡类

| 指标 | 解释 | 用法 |
| --- | --- | --- |
| RSI | 涨跌强弱 | 识别过热/超跌 |
| KDJ | 随机指标 | 短线拐点，噪声较大 |
| CCI | 偏离均值程度 | 超买超卖和趋势启动 |

## 波动类

| 指标 | 解释 | 用法 |
| --- | --- | --- |
| Bollinger Bands | 均线加减标准差 | 识别收敛、突破、回归 |
| ATR | 平均真实波幅 | 止损距离和仓位控制 |
| 历史波动率 | 收益率标准差 | 风险过滤和低波因子 |

## 量价类

| 指标 | 解释 | 用法 |
| --- | --- | --- |
| 成交额/成交量 | 市场参与度 | 过滤不可交易标的 |
| 换手率 | 筹码活跃度 | 判断热度和风险 |
| OBV | 累积量能 | 判断量价背离 |
| VWAP | 成交量加权价格 | 日内均价参考 |

## 与缠论结合

- MACD柱面积可作为背驰力度近似。
- ATR可辅助设置止损，避免固定百分比止损过粗。
- Bollinger下轨/上轨可辅助判断反转策略是否过早。
- 成交量放大但价格无法突破中枢上沿，可能表示抛压较重。

## 常见误区

- 指标越多越好。高相关指标会重复计票。
- 指标金叉就是买。金叉在震荡市容易频繁失败。
- 只优化单个指标参数。参数过拟合后样本外很容易失效。
