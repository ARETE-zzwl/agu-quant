# 回测与风控规范

## 用途

确保策略结果能解释、能复现、能比较，避免把未来函数、幸存者偏差和交易成本遗漏包装成“高收益策略”。

## 回测基础规则

| 项目 | 标准 |
| --- | --- |
| 信号时间 | 只能使用当时已经可见的数据 |
| 成交时间 | 默认下一根K线开盘成交 |
| 成本 | 至少加入佣金、印花税、滑点近似 |
| 股票池 | 记录剔除规则，如ST、停牌、低流动性 |
| 复权 | 明确前复权、后复权或不复权 |
| 调仓 | 明确频率、触发条件、最小交易单位 |
| A股规则 | T+1、涨跌停、100股整数倍 |

## 关键指标

- 总收益：策略净值相对初始净值变化。
- 年化收益：按交易日折算。
- 最大回撤：从历史高点到低点的最大跌幅。
- 夏普：单位波动下的收益。
- 胜率：盈利交易占比，不能单独评价策略。
- 盈亏比：平均盈利与平均亏损的比例。
- 交易次数：次数过少时统计不稳定。
- 换手率：决定交易成本敏感度。

## 验证流程

1. 小样本单元测试：保证策略不会崩溃。
2. 固定股票池回测：快速比较策略。
3. 滚动窗口回测：观察参数稳定性。
4. 样本外验证：用未参与调参的时间段验证。
5. 压力测试：加入更高滑点、无法成交、涨跌停。
6. 上线模拟盘：至少跟踪数周再考虑实盘。

## 风控动作

| 动作 | 触发示例 |
| --- | --- |
| 观察 | 信号不足、回测交易次数少、处于中枢内 |
| 试买 | 低仓位B1、B2、B3候选 |
| 加仓 | 持仓盈利且回抽不破关键位 |
| 减仓 | 出现S1/S2/S3、跌破中枢、市场风险升高 |
| 平仓 | 止损、结构失效、重大风险事件 |

## 禁止事项

- 用全样本调参后再报告同一全样本收益。
- 用未来财报、未来成分股、未来复权价产生信号。
- 不计交易成本比较高换手策略。
- 只展示收益率，不展示回撤和交易明细。
- 把历史最佳参数写死为“最优策略”。
