# A股交易规则

## 用途

约束模拟盘、回测成交、AI交易建议和仓位管理。任何策略如果忽略A股交易制度，回测收益都会失真。

## 交易时间

| 阶段 | 时间 |
| --- | --- |
| 开盘集合竞价 | 9:15-9:25 |
| 上午连续竞价 | 9:30-11:30 |
| 下午连续竞价 | 13:00-14:57 |
| 收盘集合竞价 | 14:57-15:00 |

## 交易单位

- 主板股票买入通常为100股整数倍。
- 卖出时不足100股的剩余部分通常应一次性卖出。
- 科创板单笔申报最低数量通常为200股。
- A股最小价格变动单位通常为0.01元。

## 涨跌幅

| 类型 | 常见涨跌幅 |
| --- | --- |
| 沪深主板普通股票 | 10% |
| 风险警示股票 | 以交易所最新规则为准 |
| 科创板、创业板 | 20% |
| 新股或恢复上市等特殊情形 | 可能无常规涨跌幅限制 |

注意：规则可能修订，系统应保留交易所规则版本检查，不要把涨跌幅写死成永久常量。

## 撮合原则

竞价交易按价格优先、时间优先撮合。集合竞价和连续竞价的成交价形成机制不同，回测里不能简单假设所有信号都能以理想价格成交。

## T+1

A股股票通常不能当日买入当日卖出。模拟盘和回测需要：

- 买入当日不可卖出该笔新仓。
- 当天已有老仓可以卖出。
- 止损信号如果发生在买入当日，只能记录风险，不能违反T+1成交。

## 系统落地

- 模拟盘：`tradingagents/paper_trade.py`
- 模拟盘页面：`web/pages/7_Paper_Trade.py`
- 回测成交：统一使用下一根K线开盘价或可成交价近似。
- 风控：涨跌停时应限制买入/卖出成交假设。
