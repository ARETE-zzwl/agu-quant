# 缠论总览

## 用途

为系统中的缠论Agent、股票监控页、策略回测和AI交易解释提供统一概念口径。缠论在社区里存在多种解释，本项目只采用可计算、可回测、可复现的子集。

## 核心口径

缠论可以拆成两部分：

- 形态学：分型、笔、线段、中枢，用来描述价格结构。
- 动力学：背驰、级别联动、买卖点，用来判断结构变化的力度和交易位置。

系统当前采用的简化链路：

```text
K线 -> 顶/底分型 -> 交替成笔 -> 三笔重叠中枢 -> 力度背驰 -> B1/B2/B3 与 S1/S2/S3 -> 策略回测
```

## 交易含义

| 结构 | 含义 | 系统用途 |
| --- | --- | --- |
| 分型 | 局部高低点 | 构造笔 |
| 笔 | 最小可操作波段 | 判断方向和力度 |
| 中枢 | 多空反复争夺区间 | 定义震荡、突破、回抽 |
| 背驰 | 价格创新高/新低但力度减弱 | 一类买卖点候选 |
| 三类买卖点 | 从左侧反转、确认到趋势延续 | 生成交易动作 |

## 工程边界

- 不把“背驰”直接等同于必然反转。
- 不追求人工画图式完美还原，优先保证同一输入得到同一输出。
- 实时信号只基于当前已经出现的K线，不使用未来数据。
- 回测采用下一根K线开盘执行，避免看见信号当根收盘后又按历史最低/最高成交。

## 常见误区

- 误区1：缠论信号等于确定盈利。实际只是一套结构化判断工具。
- 误区2：所有人画出的笔和中枢都一样。不同包含处理、成笔规则和级别选择会产生差异。
- 误区3：只看单级别。大级别决定风险区间，小级别才适合精确入场。
- 误区4：只看买点不看止损。任何买点都必须配合中枢下沿、前低或波动率止损。
